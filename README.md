# better-windows-C-toolbar
Тулбар для Windows, но в терминале, и написан на C!

# ВНИМАНИЕ
Данная программа написана при помощи ИИ, и моих мозгов (что хуже чем ИИ, поверьте мне на слово)
Программа не поддерживает кирилицу, ибо неиронки не могли нормально с этим помочь, а я сам плох в C

# Как использовать
- Сохраняем exe файл куда и где угодно
- создаём шорткаты
- в поле target добавляем путь к папке (опционально), а затем нужные расширение, для отрисовки в списке (опционально, при отсутствии просто покажет всё)

# Что оно умеет?
Программа умеет в навигацию между папками (как вверх по папкам, так и в дочерние папки, полностью под кантролем пользователя)

# Почему только для Windows??
Потому что тулбар Windows - фигня, и я хотел по лучше
